{{region}} activity conversions

Using the total number of attempts for an activity, split it up into what percentage of attempts resulted in i) Answers ii) Meaningful Interactions.

For example, to find the most efficient activity (in terms of total attempts made) for generating meaningful interactions (MI): look for the activity which has the tallest bar for MI, i.e. from the MI base to the MI top.

This only applies for activities for which we measure total attempts, total answered and total meaningful interactions. That means for this graph there will only ever be these activities:

  • Door Knocking
  • Phone Banking
  • Volunteer Recriutment Phone Calling
  • One On One
  • Stall
